# Infinity Desk — мокапы (визуальный прототип)

Это **только дизайн** — кликабельные HTML-страницы для оценки UX/UI до того как писать backend.
Никакой бэк-логики, всё статично, данные захардкожены в JS.

## Как открыть

Просто **двойной клик** по `index.html` в Finder — откроется в Chrome или Safari.

Или из терминала:
```bash
open /Users/shams/Documents/brain/infinity-pay-bot/mockups/index.html
```

## Страницы

| Файл | Что показывает | Главное |
|---|---|---|
| `index.html` | **Дашборд** — главная | KPI карточки, график тикетов за 7 дней, топ агентов, активность, urgent тикеты |
| `tickets.html` | **Тикеты** — Kanban доска | 4 колонки: Открыт / В работе / Ждём мерчанта / Закрыто. Карточки с приоритетом и агентом |
| `ticket.html` | **Один тикет** — детальный вид | Полная переписка, ответы агента/бота/мерчанта, AI-summary, действия |
| `merchants.html` | **Мерчанты** — список | Таблица 152 мерчантов с фильтрами, кодами INF-XXX, статусами тикетов |
| `reports.html` | **Отчёты** — аналитика | KPI, график 30 дней, leaderboard агентов (включая 🤖 Infinity Bot), heatmap по часам, ROI AI |

## Навигация

В сайдбаре слева — переключайся между страницами. Активная страница подсвечена синим.

В таблицах:
- **Клик по строке** мерчанта → открывает детальный тикет (демо)
- **Клик по карточке** тикета → открывает детальный тикет
- Поиск/фильтры/сортировка — визуальные (не работают, это мокап)

## Стек (для будущей реализации)

Мокап использует:
- **Tailwind CSS** через CDN (без сборки)
- **Chart.js** для графиков (line, doughnut, bar)
- **Lucide Icons** для иконок
- **Inter font** через Google Fonts

Для реального продакшена это превратится в:
- Frontend: **React + Vite + Tailwind + shadcn/ui**
- Backend: **FastAPI + PostgreSQL** (тот же `/data` volume что у бота)
- Auth: уже готово в бэкенде (`infinity-pay-dashboard/`)
- Деплой: Railway
